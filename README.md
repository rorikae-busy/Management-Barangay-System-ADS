# Management-System-Barangay


Konek sa MySQL Workbench:
1. Baguhin ung part nato sa db.php (depende sa mysql ninyo)
   
      const DB_HOST = "127.0.0.1";       makikita to pagkabukas ng mysql
   
      const DB_NAME = "barangay_db";     pangalan ng database
   
      const DB_USER = "root";            user name
   
      const DB_PASS = "MySql051312";     password ng database
   
      const DB_PORT = 3306;              port ng database


How Paganahin:

1. Open windows powershell kung saan nakalagay ung mga files nitong system the paste ito : php -S 127.0.0.1:8000 then press enter.

2. May lalabas na link kasunod (e.g. http://127.0.0.1:8000) copy that and paste sa browser.
